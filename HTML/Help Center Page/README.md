# Abstract Help Center Landing Page Clone

A frontend practice project built with HTML and CSS that recreates the Abstract Help Center landing page.

This project was created as part of a frontend development exercise using a reference design from Frontend Practice. The goal was to improve layout building, responsive design, CSS positioning, typography, spacing, and overall attention to detail by recreating a real-world website interface.

## Overview

The project replicates the visual structure of the Abstract Help Center homepage, including:

* Navigation bar
* Hero section with search functionality
* Support category cards
* Multi-column footer
* Responsive layout principles

The implementation focuses on semantic HTML and modern CSS techniques such as Flexbox and CSS Grid.

## Reference

Design reference provided by Frontend Practice:

https://www.frontendpractice.com/projects/abstract

Frontend Practice is a platform that provides real-world frontend challenges designed to help developers improve their HTML, CSS, and JavaScript skills by recreating existing websites and interfaces.

The goal of this challenge is not to create an exact pixel-perfect copy, but to practice translating a visual design into a functional webpage while developing strong frontend fundamentals.
