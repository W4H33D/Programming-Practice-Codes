# Landing Page - Front Finance

A modern, responsive landing page for a financial application called "Front Finance". This project demonstrates HTML and CSS best practices for creating a clean, professional web interface.

## 📋 Overview

This is a single-page landing page showcasing a smart credit card application. It features a clean, minimalist design with a navigation bar and a hero section that includes an eye-catching call-to-action.

## 📁 Project Structure

```
Landing Page/
├── index.html           # Main HTML file with semantic markup
├── css/
│   └── style.css        # Styling with flexbox and responsive design
├── assets/
│   └── images/          # Image assets (logo, phone mockup)
└── Reference/           # Reference materials and documentation
```

## 🎨 Features

### Layout Components

1. **Navigation Bar**
   - Logo/brand image
   - Navigation links (Features, Pricing, Help, Blog, About us)
   - Login and Sign Up buttons
   - Flexbox-based responsive layout

2. **Hero Section**
   - Attention-grabbing headline
   - Eyebrow text ("Start Saving Money")
   - Download call-to-action button
   - Hero image (smartphone mockup)
   - Split layout with text on left, image on right

### Design Highlights

- **Responsive Design**: Uses CSS flexbox for flexible layouts that work on different screen sizes
- **Color Palette**: Warm, professional colors (#faf8f4, #f8eded, #ede9f8, #1a1a1a)
- **Typography**: System font stack for optimal readability
- **Modern UI**: Rounded buttons, proper spacing, and clean aesthetics

## 🛠️ Technologies Used

- **HTML5**: Semantic markup with proper document structure
- **CSS3**: 
  - Flexbox for layout
  - CSS clamp() for responsive typography
  - Custom styling for buttons and components

## 📐 Responsive Features

- **Fluid Typography**: Uses `clamp()` for heading sizes that scale with viewport
- **Flexible Layouts**: Flexbox ensures proper element alignment across devices
- **Media-Ready**: Structure supports media queries for mobile optimization

## 🚀 Getting Started

1. Open `index.html` in your web browser
2. All assets and stylesheets are locally referenced
3. Modify `css/style.css` to customize colors, fonts, or layout
4. Replace images in `assets/images/` with your own

## 💡 Customization Tips

- **Colors**: Update color values in `style.css` (e.g., `#faf8f4`, `#1a1a1a`)
- **Navigation Links**: Modify the `<a>` href attributes in the nav section
- **Content**: Update text in the hero section and navigation
- **Images**: Replace logo and phone image paths in `assets/images/`
