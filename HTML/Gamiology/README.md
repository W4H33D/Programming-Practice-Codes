# Gamiology 🎮

A modern gaming content showcase website featuring trailers and information about popular games.

## 📖 Overview

Gamiology is a sleek, responsive web application designed to showcase gaming content including trailers, game information, and installation options. The site features a dynamic hero section with video backgrounds and a grid layout for displaying multiple games.

## 🌐 Live Site

Check out the live version of Gamiology here:
**[Visit Gamiology Live](https://w4h33d.github.io/Programming-Practice-Codes/HTML/Gamiology/)**

## ✨ Features

- **Hero Section**: Full-width video background with smooth autoplay and looping
- **Navigation Bar**: Clean, intuitive navigation with brand name and menu links
- **Game Grid**: Responsive grid layout showcasing multiple games with descriptions
- **Video Backgrounds**: Optimized video support with multiple formats (MP4, WebM)
- **Call-to-Action Buttons**: "Install Now" buttons for game downloads
- **Responsive Design**: Mobile-friendly layout that adapts to all screen sizes
- **Footer Section**: About us information
