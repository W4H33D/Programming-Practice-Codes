# Programming-Practice-Codes

Repo for practice code for different languages.

This repository contains small practice projects, exercises, and example code used for learning and experimenting with different programming languages and tools.

## Languages

- Jupyter Notebook
- Python
- C
- C++
- HTML
- CSS
- CMake

## Repository layout

Typical folders and what you'll find in them:

- `Python/` — Python scripts, modules, and small projects
- `C Lang/` — C source files and exercises
- `CPP/` or `C++/` — C++ examples and practice code
- `HTML/` and `web/` — small HTML/CSS examples
- Notebooks (`*.ipynb`) — Jupyter notebooks for experiments and data‑focused exercises

If a specific folder mentioned above isn't present, search the repository root for files matching that language.

## How to run examples

- Jupyter Notebooks: open with Jupyter or VS Code and run the cells.
- Python scripts: `python3 script.py` or `python script.py`.
- C: compile with `gcc -o prog file.c && ./prog`.
- C++: compile with `g++ -o prog file.cpp && ./prog`.
- CMake projects: create a build dir, then run `cmake ..` and `cmake --build .` from the build directory.
